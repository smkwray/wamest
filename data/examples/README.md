# Toy example data

These files are synthetic and exist only to make the repo runnable immediately.

Files:
- `toy_z1_selected_series.csv` – synthetic long-layout Z.1-style selected series extract
- `toy_h15_curves.csv` – synthetic quarter-end Treasury constant-maturity curve
- `toy_soma_holdings.csv` – synthetic quarter-end SOMA holdings snapshots
- `toy_shl_issue_mix.csv` – synthetic extracted SHL issue mix
- `toy_slt_short_long.csv` – synthetic extracted SLT short/long panel
- `toy_bank_constraint_supplement.csv` – synthetic supplement rows for affiliated-area banks plus composite reserve-access and broad-private bank perimeters
- `toy_sector_panel_ready.csv` – prebuilt toy sector panel for one test
